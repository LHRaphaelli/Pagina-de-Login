## Página de login

### Descrição

Página de login moderna com alternância entre modo escuro e claro. No modo escuro, apresenta visual futurista com efeitos neon em cyan sobre fundo preto. No modo claro, utiliza tons quentes de laranja sobre fundo bege claro. A interface possui campos para usuário e senha, link de recuperação de senha e opção de registro. O tema pode ser alternado através de um botão toggle no canto superior direito, proporcionando uma experiência visual adaptável às preferências do usuário.
